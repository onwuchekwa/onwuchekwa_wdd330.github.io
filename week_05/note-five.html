Lesson Note

Every programmer makes a mistake. Hence, bugs in codes cannot be avoided. However, a good developer finds the bugs before the users and fix them. There are many functionalities used in debugging. For example, the 'use strict' ensures that a valid JavaScript script is used. It can be used within a method or for the entire file.

Every browser has an in-built debugging console that can be use in finding and debugging codes. A bug could be handle by throwing an exception statement using the 'throw' keyword or wrap the part of code that may throw an exception within a try block. Any code placed inside a try block will pass any error objects to a catch block when an exception occurs. Any code inside a finally block will run if an exception does or does not occur.
