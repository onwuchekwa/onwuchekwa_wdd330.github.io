Lesson Note

In JavaScript, some words that perform special function are reserved. One of those keyword is this, which appears in many JavaScript frameworks such as Vue, jQuery, and others. this value changes depending on where it is called. The six different ways for assigning new values for this are:

  1. this in global context
  2. this in object construction
  3. this in object method
  4. this in simple function
  5. this in arrow function
  6. this in event listner

JavaScript objects are collections of key-value pairs that are placed inside curly braces {}. A JavaScript object is define with the following syntax: const myInfo = {name: 'Sunday', gender: 'Male', birthYear: '1981'}. The properties of an object can be accessed using either the dot notation or square bracket notation.

The Document Object Model (DOM) can be described as a representation of a HTML page as a tree of nodes. TO access the value of elements on a HTML page, the following could be used: document.getElementById(), document.getElementsByClassName(), document.getElementsByTagNames(), and document.querySelector(). You can also visit SitePoint for further study.
