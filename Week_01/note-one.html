WDD 330: Week One

Lesson Note

The easiest way to remember the name of the text field, which people use to key in short text in web application is to remember its function, input. Similarly, to remember the name of the text field that accepts long text from user is by referencing it as a textarea.

One new concept I learned this week is the use of localStorage to store data. The localStorage uses the key-value pair concept to store, retrieve, and display data. The name of the variable is the key and the value of the variable is the corresponding value to the key.

Question

  • Does the localStorage store it data in a physical storage?
  • Does the stored data persist after the browser is closed?
