Lesson Note

Functions are flexible tool to work with in JavaScript. They have built-in properties and methods, as well as custom properties and methods. They can accept other functions as parameter and even return other function. The call() and apply() methods invokes a function with the value of this, which is bound to an object provided as an argument.

There are times when a webpage will need to be updated without reloading or refreshing the page. The technique to achieve this is called Ajax. The syntax to use Ajax is: 

    fetch(request)
    .then(
        // do something with the request
    )
    .catch(
        // handle any errors
    )

Question

  • What are other ways to use Ajax except using the Fetch API?
