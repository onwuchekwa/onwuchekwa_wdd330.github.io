Lesson Note

CSS Transform and Transition properties are ways to manipulate or animate the appearance of text, images, and other objects on a webpage.

With the transform property, objects and text on a webpage can be rotated, translated, scaled, and/or skewed. The syntax for transform is: 

    .image {
        transform: rotate(10deg) translateX(40px) scale(1.5) skew(15deg, 4deg);
    }
The appearance of elements could be manipulated using the transform functions. Its property is one or more functions, separated by spaces as depicted in the above. The webpage will the render the object in the order in which they are apecified.

CSS transition is used to animate objects in a webpage using animation and keyframes properties.

HTML's canvas API allows web designers to draw anything they can imagine. This is powered by JavaScript. The Cancas API is supported by all modern browsers. The syntax to implement canvas is: 

    <canvas id="canvasId" class="canvasClass">
      Sorry! Your browser does not support Canvas
    </canvas>
