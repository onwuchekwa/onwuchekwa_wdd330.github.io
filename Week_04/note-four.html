Lesson Note

Forms allow the visitors of a website to enter data into the browser. The element <form> contains controls such as input fields, dropdowns, and buttons. Forms can also trigger a number of events such as click, focus, submit, reset, and so forth.

Some of the common types of form control are:

  • <input>
  • <select>
  • <button>
  • <textarea>
Form validation help developers check whether a user enters a valid input. We are required to implement both the client-side and serveriside validations.

Object-oriented programming (OOP) is pattern objects in our codes after a real-life object. This can be done through encapsulation, polymorphism, and inheritance.

JavaScript libraries provide a way that make programming easier. However, programmers need to decide whether using JavaScript libraries is the best. Some examples of the libraries are jQuery, Lodash.

Question

  • Why should we implement both the client-side and server-side validation?
  • When should we consider using a JavaScript library?
